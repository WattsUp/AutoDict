====================================
Welcome to AutoDict's documentation!
====================================

AutoDict is a Python dictionary that automatically adds children dictionaries as
necessary. JSONAutoDict wraps a file and AutoDict to automatically save when
exiting a context. Files use json based serialization and deserialization.

The latest version is AutoDict |release|, released |release_date|.  It is
supported on:

* Python versions 3.7 through 3.10.
